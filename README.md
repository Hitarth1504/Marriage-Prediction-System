# Marriage Prediction System

## Project Description
This project predicts the type of marriage using Machine Learning techniques.
The dataset contains social and personal factors such as education, religion,
parental approval, dowry, and marital satisfaction.

The project uses Logistic Regression for prediction.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Dataset
File: marriage_data_india.csv  

Attributes used:
- Age at Marriage
- Gender
- Education Level
- Caste Match
- Religion
- Parental Approval
- Dowry Exchanged
- Marital Satisfaction
- Years Since Marriage
- Spouse Working
- Inter-Caste Marriage
- Inter-Religion Marriage

## Project Workflow
1. Data loading
2. Data cleaning
3. Label encoding
4. Train-test split
5. Model training
6. Accuracy evaluation

## Algorithm Used
- Logistic Regression
